# OpenEMR Full EHR Integration

Complete MCP plugin for OpenEMR, the world's most popular open-source EHR. Connects any MCP-compatible AI assistant to an OpenEMR instance for patient records, scheduling, e-prescriptions, lab integration, and HIPAA ASC X12 billing support. Works in mock mode for evaluation without OpenEMR installation.
