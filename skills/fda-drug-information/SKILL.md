# FDA Drug Information & Adverse Events

Access comprehensive FDA drug databases including adverse events (FAERS), drug labels, NDC directory, recalls, shortages, and 510(k) device clearances via openFDA API.

> **Status:** In Development — Implementation in progress. Core functionality defined, full feature set coming soon.

## Quick Install

```bash
npx skills add gitjfmd/open-medical-skills --skill fda-drug-information
```

## What It Does

- Access comprehensive FDA drug databases including adverse events (FAERS), drug labels, NDC directory, recalls, shortages, and 510(k) device clearances via openFDA API.

## Clinical Use Cases

- Drug interaction screening
- Medication safety verification
- Formulary management and alternatives

## Safety & Evidence

- **Safety Classification:** Safe — This skill provides informational support and does not directly influence clinical decisions.
- **Evidence Level:** High — Based on systematic reviews, randomized controlled trials, or authoritative clinical guidelines.
