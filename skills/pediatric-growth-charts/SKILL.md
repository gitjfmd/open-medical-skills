# Pediatric Growth Chart Analyzer

Interpret pediatric growth measurements using CDC and WHO growth charts. Plots height, weight, BMI, and head circumference percentiles for age and gender.

> **Status:** In Development — Implementation in progress. Core functionality defined, full feature set coming soon.

## Quick Install

```bash
npx skills add gitjfmd/open-medical-skills --skill pediatric-growth-charts
```

## What It Does

- Interpret pediatric growth measurements using CDC and WHO growth charts. Plots height, weight, BMI, and head circumference percentiles for age and gender.

## Clinical Use Cases

- Age-appropriate clinical assessments
- Growth and developmental monitoring
- Pediatric dosing calculations
