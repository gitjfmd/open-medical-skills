# Interactive Anatomy Learning Assistant

Interactive anatomy education tool with 3D anatomical references, clinical correlations, and self-assessment questions. Covers all major organ systems.

## Quick Install

```bash
npx skills add gitjfmd/open-medical-skills --skill medical-anatomy-tutor
```

## What It Does

- Interactive anatomy education tool with 3D anatomical references, clinical correlations, and self-assessment questions. Covers all major organ systems.

## Clinical Use Cases

- Medical knowledge assessment and review
- Case-based learning and clinical reasoning
- Board exam preparation and practice
